## Install Instructions:
### Linux:
#### Arch:
Cmake:

```
pacman -S cmake
```
Git:
```
pacman -S git
```

#### Debian/Ubuntu
Cmake:
```
apt install cmake
```
Git:
```
apt install cmake
```
#### NeoVim
change :make command to use bear to gen compile_commands.json
```
set makeprg=bear\ --\ make
```
